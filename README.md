# cluster-pipeline
## Stages
- Create Cluster
- Create a kubectl configuration file
### Create Cluster
Cluster is created through "eksctl create cluster"
### Create a kubectl configuration file
 kubectl is comfigured through "aws eks --region ap-southeast-2 update-kubeconfig --name capstonecluster"
